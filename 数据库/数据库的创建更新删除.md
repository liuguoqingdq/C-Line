**1.创建数据库**
```SQL
//方式1
CREATE DATABASE IF NOT EXISTS test1;
SHOW test1

//方式2
CREATE DATABASE test2 CHARACTER SET 'gbk'


//切换数据库
USE test2;

//查看表
SHOW TABLES;

//查看当前使用数据库
SELECT DATABASE FROM DAUL;
```

**2.修改数据库**
```SQL
ALTER DATABASE 数据库名 CHARACTER SET '字符集'；
```

**3.删除库**
```SQL
DROP DATABASE IF EXISTS 数据库名;
```
