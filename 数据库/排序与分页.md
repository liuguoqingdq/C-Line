### 排序
**使用ORDER BY对搜索到的字段进行排序(默认从低到高升序[ASC])使用降序是DESC

```SQL
SELECT id,name,salary 
FROM employee
ORDER BY salary;//升序

SELECT id,name,salary
FROM employee
ORDER BY salary DESC;//降序


SELECT id,name,salary*12 AS anual_sal
FROM employee
ORDER BY anual_sal;//可以对别名进行排序，只能在ORDER BY中使用，不能在WHERE中使用
```
**！！！WHERE需要声明在ORDER BY之前，FROM之后

**二级排序
```SQL
SELECT id,name,department_id AS dep_id,salary
FROM employee
ORDER BY dep_id DESC,salary ASC;//department_id降序，每个dep_id对应的salary段升序
```


### 分页
使用LIMIT分页
**1.每页显示20条记录，从第一页开始**
```SQL
SELECT id,name,salary*12 AS anual_sal
FROM employee
ORDER BY anual_sal
LIMIT 0,20;
```

**2.每页显示20条记录，从第二页开始**
```SQL
SELECT id,name,salary*12 AS anual_sal
FROM employee
ORDER BY anual_sal
LIMIT 20,20;
```

**3.每页显示20条记录，从第二页开始**
```SQL
SELECT id,name,salary*12 AS anual_sal
FROM employee
ORDER BY anual_sal
LIMIT 40,20;
```

**4.表里有107条数据，显示32与33条数据**
```SQl
SELECT id,name,salary*12 AS anual_sal
FROM employee
ORDER BY anual_sal
LIMIT 31,2;


//8.0新特性
SELECT id,name,salary*12 AS anual_sal
FROM employee
ORDER BY anual_sal
LIMIT 2 OFFSET 31;
```
